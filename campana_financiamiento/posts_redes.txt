POST 1 - LANZAMIENTO
Estamos construyendo Save Swimmer, un prototipo de seguridad para nadadores de aguas abiertas.

La idea no es crear un reloj deportivo. Buscamos desarrollar un wearable dorsal que registre el comportamiento corporal del nadador y ayude a interpretar si su nado mantiene ritmo, orientacion y movimiento funcional.

Ya tenemos firmware, app de campo, demo visual y primeras pruebas con datos reales. Ahora necesitamos apoyo para avanzar la version Lite de campo: sensor de agua, GPS de prueba, carcasa, energia y validacion en playa.

POST 2 - PROBLEMA
En aguas abiertas, muchas veces el entrenador mira desde la playa, la familia espera noticias y el nadador esta lejos de la orilla.

Save Swimmer busca convertir datos corporales en informacion entendible:
- esta nadando
- esta dentro de zona
- mantiene ritmo
- bajo el rendimiento
- se detuvo
- regreso a playa

Estamos en etapa de prototipo y buscamos apoyo para financiar la siguiente fase.

Campana StartFund:
https://startfund.pe/proyectos/save-swimmer-seguridad-para-nadadores-de-aguas-abiertas.php


POST 3 - AVANCE
Save Swimmer ya no es solo una idea.

Hoy el prototipo puede:
- leer movimiento corporal
- conectarse por Bluetooth
- grabar sesiones en microSD
- enviar datos a una app Android
- visualizar brazada, rotacion y ritmo
- simular la experiencia final para nadador, coach, familia y evento

El siguiente paso es hacerlo mas robusto para pruebas reales en playa.


POST 4 - LLAMADO DIRECTO
Buscamos apoyo para construir el prototipo Save Swimmer Lite.

Meta inicial sugerida: S/ 5,000

Esto ayudara a cubrir componentes, carcasa de prueba, sensor de agua, GPS/GNSS, baterias, materiales de sellado, app y pruebas de campo.

Cada aporte suma. Cada prueba nos acerca a un dispositivo mas seguro, compacto y util para nadadores de aguas abiertas.


POST 5 - PARA ENTRENADORES
Entrenador: imagina ver a tus nadadores activos en un mapa, dentro de un geocerco, con estados simples como ritmo estable, ritmo bajo, sin avance o fuera de zona.

Save Swimmer busca darle al coach una lectura rapida y accionable sin obligarlo a interpretar datos crudos.

Estamos construyendo el prototipo y buscamos aliados para la fase de pruebas.


POST 6 - PARA FAMILIA
Para la familia no hacen falta graficos tecnicos. Hace falta saber algo simple:

Esta en el agua.
Esta nadando.
Esta cerca.
Ya salio.

Save Swimmer busca transformar sensores y movimiento en estados claros para acompanamiento y seguridad.
