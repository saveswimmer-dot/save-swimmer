# Save Swimmer Lite - mockup carcasa V001

Fecha: 2026-07-12  
Estado: muestra conceptual para revisar volumen, postura y conversacion con proveedor de impresion 3D.

## Archivos

- `SS-CARCASA-LITE-MOCKUP-V001.scad`: CAD parametrico editable.
- `SS-CARCASA-LITE-MOCKUP-V001.stl`: volumen imprimible aproximado.
- `SS-CARCASA-LITE-MOCKUP-V001.dxf`: contorno 2D superior con marcas de correa/tornillos.
- `SS-CARCASA-LITE-MOCKUP-V001.obj`: referencia simple para visor 3D.
- `SS-CARCASA-LITE-MOCKUP-V001.png`: lamina visual para enviar o conversar.

## Supuestos usados

- Cuerpo dorsal aproximado: 95 x 55 x 22 mm.
- Canal de correa/velcro: 22 mm.
- Base con curva suave para apoyar sobre gorra, boya personal, neopreno o soporte dorsal.
- Zona superior reservada para antena/GPS/LTE.
- Boton/LED en zona separada para operacion rapida.

## Importante

Esto no es una carcasa final estanca. Sirve para:

- validar si el volumen molesta al nadador;
- estimar posicion en cuerpo/correa;
- conversar con impresor 3D o disenador industrial;
- comparar ubicacion de antena, boton, LED y paso de correa.

Antes de producto final falta definir PCB real, bateria, conectores, sellado, carga, tornillos, junta, flotabilidad, impacto, salinidad y ubicacion exacta de antenas.
